🔧 TROUBLESHOOTING
   Issue: Server not starting
   Fix:   Check if port 3000 is available
          Kill existing process: kill $(lsof -t -i:3000)

   Issue: API keys not working
   Fix:   Verify .env.local file exists and has correct keys
          Restart dev server after changing .env.local

   Issue: Firebase auth errors
   Fix:   Check Firebase Console → Authentication → Sign-in methods
          Ensure authorized domains include localhost

   Issue: Maps not loading
   Fix:   Verify GOOGLE_MAPS_API_KEY in .env.local
          Check API is enabled in Google Cloud Console
          Ensure billing is set up for Maps API
